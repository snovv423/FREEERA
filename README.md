# FREEERA

Сайт лимитированных капсул FREEERA. Статический сайт (HTML), без сервера.

## Файлы
- `index.html` — главная (выбор капсулы)
- `freeera-ronin.html` — капсула RONIN, детское BJJ ги
- `freeera-stone-flame.html` — капсула STONE & FLAME, футболка

Все три файла должны лежать рядом, в корне репозитория. Имена менять нельзя — иначе ссылки между страницами сломаются.

Заявки с сайта открываются в Telegram (@killkelly) с уже заполненным заказом.
